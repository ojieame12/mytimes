# Product

## Register
brand

## Users
Organizers needing to quickly set up a batch of 10-25 interview slots.
Participants needing to quickly claim a time from a shared link without an account.

## Product Purpose
mytimes turns a fixed list of interview times into a public booking link and a private organizer admin link. It optimizes for two short-lived jobs: creating an event in <4 minutes, and claiming a slot in <30 seconds.

## Brand Personality
Tangible, focused, compact, analog-inspired (letterpress, wax-stamped, warm paper). Not SaaS-y or playful.

## Anti-references
Typical bloated calendar SaaS (Calendly, etc.). No recurring rules, no multi-organizer flows, no nested cards, no generic SaaS dashboards.

## Design Principles
- Speed over features
- Tangible affordances
- Privacy by default
- One primary action per screen

## Accessibility & Inclusion
WCAG AA baseline, high contrast for text, clear focus states for fast keyboard navigation, mindful of reduced motion.
